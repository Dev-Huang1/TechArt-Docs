| Name | Description |
| ---- | ----------- |
| data-callback | 必填，用户提交成功响应时执行的回调函数的名称。一个验证码令牌将传递给您的回调 |
| data-time | 必填，Token 的有效时间以及 Captcha 的过期时间（单位：毫秒）|
| data-lang | 可选，这是自定义语言，[目前支持的语言](https://docs.xyehr.cn/zh-cn/docs/one-captcha/more/supported-language/) |
